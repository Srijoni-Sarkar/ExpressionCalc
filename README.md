# Expression Calculator in C

## Description
This is a simple command-line calculator written in C that evaluates arithmetic expressions. It converts *infix expressions* to *postfix* and then evaluates the result using a stack.  

### Features
- Supports *multi-digit numbers*  
- Handles *negative numbers*  
- Supports *parentheses* for grouping  
- Respects *operator precedence* (+, -, *, /)  
- Shows *postfix expression* and *evaluation result*  

## Files
- main.c : Handles input/output  
- eval.c  : Contains infixToPostfix() and evalPostfix() functions  
- tests.txt : Sample inputs and expected outputs (optional)  
- .gitignore : To ignore compiled files (*.exe)
